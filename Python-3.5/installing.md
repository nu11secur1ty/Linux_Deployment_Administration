# Download Python 3.5 version using the following commands.
```bash
cd /usr/src
wget https://www.python.org/ftp/python/3.5.5/Python-3.5.5.tgz
```
# Now extract the downloaded archive on your system.
```bash
sudo tar xzf Python-3.5.5.tgz
```
