# Installing geckodriver:
```bash
wget https://github.com/mozilla/geckodriver/releases/download/v0.20.1/geckodriver-v0.20.1-linux64.tar.gz
tar -xvf geckodriver-v0.20.1-linux64.tar.gz
cp geckodriver /usr/bin/
```
