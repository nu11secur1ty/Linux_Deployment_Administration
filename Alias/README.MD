# Commands

- - - Available alias options

```bash
alias egrep='egrep --color=auto'
alias fgrep='fgrep --color=auto'
alias grep='grep --color=auto'
alias l='ls -CF'
alias la='ls -A'
alias ll='ls -alF'
alias ls='ls --color=auto'
```
- - - Example

- Create file `koko`

```bash
touch /var/log/koko
```
